´# 🎯 Crozzhair

> **Precision Crosshair Overlay for Windows** – Customize, configure, and display dynamic crosshairs on your screen with ease.

---

## ✨ Features

- 🎨 **Fully Customizable Crosshairs** – Adjust colors, sizes, styles, and opacity
- ⌨️ **Hotkey Binding** – Toggle overlays with custom keyboard shortcuts
- 💾 **Preset System** – Save and load multiple crosshair configurations
- 🖥️ **System Tray Integration** – Minimize to tray for quick access
- ⚡ **Lightweight & Fast** – Smooth performance and instant response
- 🔒 **Single Instance** – Prevents multiple instances from running simultaneously
- 🎮 **Gaming-Ready** – Optimized for competitive gaming and precision tasks

---

## 🚀 Installation

### Requirements
- **Windows 10/11** (x64)

### Download & Install

1. Download the latest installer from [Releases](../../releases)
2. Run `Crozzhair-Setup.exe`
3. Follow the installation wizard
4. Launch from Start Menu or desktop shortcut

---

## 🎮 Usage

1. **Launch Crozzhair** from the Start Menu or desktop shortcut
2. **Configure Your Crosshair** in the Control Panel:
   - Choose color, size, and style
   - Set transparency and line thickness
3. **Bind a Hotkey** to toggle the overlay on/off
4. **Save Presets** for quick switching between configurations
5. **Minimize to Tray** for background operation

### Keyboard Shortcuts
- **Default Toggle**: `Ctrl + Alt + X` (customizable)
- **Right-click Tray Icon**: Access menu and settings

---

## ⚙️ Configuration

### Settings File
Crozzhair stores user settings in:
```
%APPDATA%\Crozzhair\settings.json
```

### Customization Options
- **Crosshair Style**: Dot, crosshair, circle, custom shapes
- **Color**: RGB picker with hex support
- **Size**: 5-100 pixels
- **Opacity**: 0-100%
- **Line Thickness**: 1-10 pixels
- **Hotkey**: Any keyboard combination

## 🐛 Troubleshooting

| Issue | Solution |
|-------|----------|
| **Overlay not showing** | Ensure the application has focus; check hotkey binding |
| **Crashes on startup** | Verify Windows 10/11 x64; reinstall the application |
| **Settings not saving** | Check `%APPDATA%\Crozzhair` folder permissions |
| **Hotkey conflicts** | Rebind to an unused key combination |
